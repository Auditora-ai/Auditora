## BUG-025: Hardcoded Spanish strings in `node-display-config.ts` and `NodeContextPanel.tsx`

**Severity:** Low  
**Component:** `apps/saas/modules/meeting/lib/node-display-config.ts` + `apps/saas/modules/process-library/components/workspace/NodeContextPanel.tsx`  
**Found by:** QA Agent #05  
**Date:** 2026-04-02  
**Status:** 🔴 OPEN

---

### Steps to Reproduce

1. Navigate to the process library workspace view (process detail, node panel)
2. Click on a Start Event or End Event node in the BPMN diagram
3. Observe the NodeContextPanel shows Spanish text for non-Spanish users

### Expected Behavior

UI labels should use i18n translation keys, not hardcoded Spanish strings.

### Actual Behavior

**File 1: `node-display-config.ts`** (lines 19-20):

```typescript
"bpmn:StartEvent": { ..., label: "Inicio" },    // ← hardcoded Spanish
"bpmn:EndEvent":   { ..., label: "Fin" },        // ← hardcoded Spanish
"bpmn:Task":       { ..., label: "Tarea" },      // ← hardcoded Spanish
"bpmn:UserTask":   { ..., label: "Tarea Usuario" }, // ← hardcoded Spanish
```

All BPMN node type labels are hardcoded Spanish.

**File 2: `NodeContextPanel.tsx`** (lines 145-146):

```typescript
{element.type.includes("Start") && "Este evento inicia el proceso."}
{element.type.includes("End") && "Este evento finaliza el proceso."}
```

Hardcoded Spanish tooltip text in the workspace sidebar.

### Code References

- `apps/saas/modules/meeting/lib/node-display-config.ts` — lines 19-25 (all label values)
- `apps/saas/modules/process-library/components/workspace/NodeContextPanel.tsx` — lines 145-146

### Suggested Fix

1. For `node-display-config.ts`: Since this is a config object (not a React component), labels should be translation keys rather than rendered strings. Or export a hook that resolves the labels using `useTranslations()`.

2. For `NodeContextPanel.tsx`: Replace hardcoded strings with `t()` calls:
```typescript
{element.type.includes("Start") && t("processLibrary.workspace.startEventDescription")}
{element.type.includes("End") && t("processLibrary.workspace.endEventDescription")}
```

### Regression Risk

Low — cosmetic/UX issue only. Affects non-Spanish users who see Spanish labels in the BPMN workspace node panel. No functional breakage.
